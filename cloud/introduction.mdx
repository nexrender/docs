Welcome to the official documentation for Nexrender Cloud – the hosted API-based solution for automating Adobe After Effects rendering at scale.


## Nexrender Basics
### What is Nexrender Cloud?
Nexrender Cloud is a managed version of [Nexrender OSS](https://github.com/inlife/nexrender), designed for:

- Developers who automate build video customization and rendering pipelines
- Studios and agencies who need batch rendering in the cloud with guaranteed reliability
- Product teams that need frictionless video rendering and deep customization to power user experiences

### Key Features
- Fully managed rendering infrastructure – No local setup, render farm, or plugins required
- Scalable cloud backend – Automatically handles high-volume render spikes and concurrent job loads
- Fast job submission via REST API
- .aep, .zip, and .mogrt support
- Template analytics and introspection
- Dynamic asset injection – Replace text, images, videos, audio, and nested layers via declarative JSON
- Upload and persist custom font files to ensure brand consistency across renders
- After Effects expressions support
- Secure output delivery
